# Next.js Layout Example

This is the demo repo for the blog article on [Next.js 13 Layouts by Example](https://blog.mechanicalrock.io/2022/10/27/next-js-layouts-by-example.html)

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
